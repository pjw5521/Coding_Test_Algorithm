# 04. 구현 

## 구현이란?
- 머릿속에 있는 알고리즘을 소스코드로 바꾸는 과정
- 모든 범위의 코딩 테스트 문제 유형을 포함하는 개념 
- 프로그래밍 문법을 정확하게 숙지하지 못했거나, 라이브러리 사용 경험이 부족하면 불리
- 완전 탐색 : 모든 경우의 수를 주저 없이 다 계산하는 해결방법
- 시뮬레이션 : 문제에서 제시한 알고리즘을 한단계씩 차례로 수행하는 해결방법 

## 메모리 제약 사항
- python 리스트 크기
- 채점환경
